\doxysubsection{Detailed Description}
Data structure for a material. 

\mbox{\hyperlink{class_material}{Material}} data is stored using a key-\/value structure. A single key-\/value pair is called a \textquotesingle{}material property\textquotesingle{}. C++ users should use the provided member functions of \mbox{\hyperlink{structai_material}{ai\+Material}} to process material properties, C users have to stick with the ai\+Material\+Get\+XXX family of unbound functions. The library defines a set of standard keys (AI\+\_\+\+MATKEY\+\_\+\+XXX).
