\doxysubsection{Detailed Description}
\subsubsection*{template$<$bool Swap\+Endianness = false, bool Runtime\+Switch = false$>$\newline
class Assimp\+::\+Stream\+Writer$<$ Swap\+Endianness, Runtime\+Switch $>$}
Wrapper class around \mbox{\hyperlink{class_assimp_1_1_i_o_stream}{IOStream}} to allow for consistent writing of binary data in both little and big endian format. Don\textquotesingle{}t attempt to instance the template directly. Use Stream\+Writer\+LE to write to a little-\/endian stream and Stream\+Writer\+BE to write to a BE stream. Alternatively, there is Stream\+Writer\+Any if the endianness of the output stream is to be determined at runtime.
