ai\+Primitive\+Type\+\_\+\+NGONEncoding\+Flag&A flag to determine whether this triangles only mesh is NGON encoded. NGON encoding is a special encoding that tells whether 2 or more consecutive triangles should be considered as a triangle fan. This is identified by looking at the first vertex index. 2 consecutive triangles with the same 1st vertex index are part of the same NGON.

At the moment, only quads (concave or convex) are supported, meaning that polygons are \textquotesingle{}seen\textquotesingle{} as triangles, as usual after a triangulation pass.

To get an NGON encoded mesh, please use the ai\+Process\+\_\+\+Triangulate post process.

At the moment, only quads (concave or convex) are supported, meaning that polygons are \textquotesingle{}seen\textquotesingle{} as triangles, as usual after a triangulation pass.

To get an NGON encoded mesh, please use the ai\+Process\+\_\+\+Triangulate post process.
