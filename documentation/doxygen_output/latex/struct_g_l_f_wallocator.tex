\doxysubsection{Detailed Description}
Custom heap memory allocator. 

This describes a custom heap memory allocator for GLFW. To set an allocator, pass it to \mbox{\hyperlink{group__init_ga0f4c8556a21d9e774a0ba0332da1748e}{glfw\+Init\+Allocator}} before initializing the library.
