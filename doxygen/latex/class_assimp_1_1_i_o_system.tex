\doxysubsection{Detailed Description}
CPP-\/\+API\+: Interface to the file system. 

Derive an own implementation from this interface to supply custom file handling to the importer library. If you implement this interface, you also want to supply a custom implementation for \mbox{\hyperlink{class_assimp_1_1_i_o_stream}{IOStream}}.
