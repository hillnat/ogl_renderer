\doxysubsection{Detailed Description}
FOR IMPORTER PLUGINS ONLY\+: Simple exception class to be thrown if an unrecoverable error occurs while importing. Loading APIs return nullptr instead of a valid \mbox{\hyperlink{structai_scene}{ai\+Scene}} then. ~\newline
