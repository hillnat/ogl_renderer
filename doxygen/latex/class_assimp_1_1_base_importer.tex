\doxysubsection{Detailed Description}
FOR IMPORTER PLUGINS ONLY\+: The \mbox{\hyperlink{class_assimp_1_1_base_importer}{Base\+Importer}} defines a common interface for all importer worker classes.

The interface defines two functions\+: \mbox{\hyperlink{class_assimp_1_1_base_importer_a13588d3396ba5b7ed1f1cb46e0945cfd}{Can\+Read()}} is used to check if the importer can handle the format of the given file. If an implementation of this function returns true, the importer then calls \mbox{\hyperlink{class_assimp_1_1_base_importer_a2f5938785f722ebde616a30160bccde1}{Read\+File()}} which imports the given file. Read\+File is not overridable, it just calls \mbox{\hyperlink{class_assimp_1_1_base_importer_ac67d9f5ceb26353d27d6be06cccad398}{Intern\+Read\+File()}} and catches any Import\+Error\+Exception that might occur.

 The output scene must meet the following requirements\+:~\newline
 
\begin{DoxyItemize}
\item At least a root node must be there, even if its only purpose is to reference one mesh. 
\item \mbox{\hyperlink{structai_mesh_a99d66ac0a444068c1b252b30265cbf53}{ai\+Mesh\+::m\+Primitive\+Types}} may be 0. The types of primitives in the mesh are determined automatically in this case. 
\item the vertex data is stored in a pseudo-\/indexed \char`\"{}verbose\char`\"{} format. In fact this means that every vertex that is referenced by a face is unique. Or the other way round\+: a vertex index may not occur twice in a single \mbox{\hyperlink{structai_mesh}{ai\+Mesh}}. 
\item \mbox{\hyperlink{structai_animation_ae2bbc49320b4b75c05e23e0ab704ece7}{ai\+Animation\+::m\+Duration}} may be -\/1. \mbox{\hyperlink{namespace_assimp}{Assimp}} determines the length of the animation automatically in this case as the length of the longest animation channel. 
\item \mbox{\hyperlink{structai_mesh_a4d2b15fe360d7c6f45c28dced096510a}{ai\+Mesh\+::m\+Bitangents}} may be nullptr if tangents and normals are given. In this case bitangents are computed as the cross product between normal and tangent. 
\item There needn\textquotesingle{}t be a material. If none is there a default material is generated. However, it is recommended practice for loaders to generate a default material for yourself that matches the default material setting for the file format better than \mbox{\hyperlink{namespace_assimp}{Assimp}}\textquotesingle{}s generic default material. Note that default materials {\itshape should} be named AI\+\_\+\+DEFAULT\+\_\+\+MATERIAL\+\_\+\+NAME if they\textquotesingle{}re just color-\/shaded or AI\+\_\+\+DEFAULT\+\_\+\+TEXTURED\+\_\+\+MATERIAL\+\_\+\+NAME if they define a (dummy) texture.  
\end{DoxyItemize}If the AI\+\_\+\+SCENE\+\_\+\+FLAGS\+\_\+\+INCOMPLETE-\/\+Flag is {\bfseries{not}} set\+:

\item there may be no meshes with 0 vertices or faces
