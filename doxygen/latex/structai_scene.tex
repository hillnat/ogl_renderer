\doxysubsection{Detailed Description}
The root structure of the imported data.

Everything that was imported from the given file can be accessed from here. Objects of this class are generally maintained and owned by \mbox{\hyperlink{namespace_assimp}{Assimp}}, not by the caller. You shouldn\textquotesingle{}t want to instance it, nor should you ever try to delete a given scene on your own.
