\doxysubsection{Detailed Description}
A skeleton represents the bone hierarchy of an animation. 

Skeleton animations can be described as a tree of bones\+: root $\vert$ node1 / \textbackslash{} node3 node4 If you want to calculate the transformation of node three you need to compute the transformation hierarchy for the transformation chain of node3\+: root-\/\texorpdfstring{$>$}{>}node1-\/\texorpdfstring{$>$}{>}node3 Each node is represented as a skeleton instance.
