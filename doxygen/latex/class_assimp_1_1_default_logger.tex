\doxysubsection{Detailed Description}
CPP-\/\+API\+: Primary logging facility of \mbox{\hyperlink{namespace_assimp}{Assimp}}. 

The library stores its primary \mbox{\hyperlink{class_assimp_1_1_logger_a17f4cbc042c26935e16eb204ca77c2c1}{Logger}} as a static member of this class. \mbox{\hyperlink{class_assimp_1_1_default_logger_a4b6ac25541ce63973f7fb0bf1b8bcb44}{get()}} returns this primary logger. By default the underlying implementation is just a \#\+Null\+Logger which rejects all log messages. By calling \mbox{\hyperlink{class_assimp_1_1_default_logger_aeba2d768f7549fb218bcce1245e88e1c}{create()}}, logging is turned on. To capture the log output multiple log streams (\#\+Log\+Stream) can be attach to the logger. Some default streams for common streaming locations (such as a file, std\+::cout, Output\+Debug\+String()) are also provided.

If you wish to customize the logging at an even deeper level supply your own implementation of \mbox{\hyperlink{class_assimp_1_1_logger_a17f4cbc042c26935e16eb204ca77c2c1}{Logger}} to \mbox{\hyperlink{class_assimp_1_1_default_logger_a9daba548026045b99813c760c2842ed2}{set()}}. \begin{DoxyNote}{Note}
The whole logging stuff causes a small extra overhead for all imports. 
\end{DoxyNote}
