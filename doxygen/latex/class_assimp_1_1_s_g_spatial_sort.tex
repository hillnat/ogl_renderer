\doxysubsection{Detailed Description}
Specialized version of \mbox{\hyperlink{class_assimp_1_1_spatial_sort}{Spatial\+Sort}} to support smoothing groups This is used in by the 3DS, ASE and LWO loaders. 3DS and ASE share their normal computation code in Smoothing\+Groups.\+inl, the LWO loader has its own implementation to handle all details of its file format correctly.
