\doxysubsection{Detailed Description}
\subsubsection*{template$<$bool Swap\+Endianness = false, bool Runtime\+Switch = false$>$\newline
class Assimp\+::\+Stream\+Reader$<$ Swap\+Endianness, Runtime\+Switch $>$}
Wrapper class around \mbox{\hyperlink{class_assimp_1_1_i_o_stream}{IOStream}} to allow for consistent reading of binary data in both little and big endian format. Don\textquotesingle{}t attempt to instance the template directly. Use Stream\+Reader\+LE to read from a little-\/endian stream and Stream\+Reader\+BE to read from a BE stream. The class expects that the endianness of any input data is known at compile-\/time, which should usually be true (\mbox{\hyperlink{class_assimp_1_1_base_importer_afc318aa0eb85b73f501156a50f3eb703}{Base\+Importer\+::\+Convert\+To\+UTF8}} implements runtime endianness conversions for text files).
