\doxysubsection{Detailed Description}
Intermediate description a vertex with all possible components. Defines a full set of operators, so you may use such a \textquotesingle{}\mbox{\hyperlink{struct_assimp_1_1_vertex}{Vertex}}\textquotesingle{} in basic arithmetic. All operators are applied to {\itshape all} vertex components equally. This is useful for stuff like interpolation or subdivision, but won\textquotesingle{}t work if special handling is required for some vertex components.
